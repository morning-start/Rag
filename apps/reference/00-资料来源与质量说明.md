## 搜索策略

### 搜索词模版

```
英文搜索词：
[RAG topic] tutorial 2025 2026 complete guide
[RAG topic] best practices production
site:learn.microsoft.com RAG tutorial
site:github.com RAG from scratch tutorial

中文搜索词：
RAG 教程 完整指南 2026
生产级 RAG 落地 最佳实践
RAG 系统设计 核心组件
```

### 质量判断标准

1. **时效性**：优先 2025-2026 年的资料
2. **结构完整性**：有清晰的目录结构、代码示例
3. **作者背景**：有实际工程经验的作者
4. **引用数据**：有数据来源、标注参考
5. **社区验证**：GitHub Star 数、文章点赞/评论

## 建议的搜索优先级

1. 先搜官方文档（Microsoft / LangChain / LlamaIndex）
2. 再搜 GitHub 完整教程
3. 补充行业数据（Gartner / Deloitte / McKinsey）
4. 参考中文社区文章（补充实践细节和避坑经验）
