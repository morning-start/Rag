## 核心内容摘要

CSDN 全面的 RAG 优化指南，覆盖从数据加载到生成的完整链路优化方案，包含检索环节深度优化、Prompt增强与生成环节优化、工程化性能优化等。

## 关键数据与论点

### 检索质量优化（对应 Ch6）
- 分块策略：递归字符分块是生产默认选择；语义分块质量最高但成本高
- 元数据过滤：按时间/来源/类型过滤可将不相关结果减少 60%+
- MMR 多样性控制 lambda_mult=0.3-0.5 平衡相关性和多样性
- 上下文压缩：长文档先摘要再注入可提升信噪比翻倍

### 生成质量优化（对应 Ch7）
- Prompt 工程六大要素：角色设定、核心规则、格式约束、引用溯源、异常处理、安全护栏
- 幻觉防控参数锁死：Temperature ≤ 0.3，Top-P ≤ 0.9
- 引用强制约束：要求每个事实性表述标注来源，从生成环节杜绝无依据编造
- 动态 Prompt：基于检索结果质量动态调整指令，不足时触发拒答或补充检索

### 工程化性能优化（对应 Ch9）
- 缓存策略：热门查询缓存命中率可达 40%-60%（客服问答场景）
- 批量索引：异步批量 Embedding 索引速度提升 3-5x
- LLM 推理加速：4-bit 量化 + vLLM 批处理，吞吐量提升 3-5x
