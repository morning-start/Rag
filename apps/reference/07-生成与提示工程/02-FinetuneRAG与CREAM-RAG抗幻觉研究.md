# Finetune-RAG 与 CREAM-RAG 抗幻觉研究

> 来源: arXiv 2505.10792 (Finetune-RAG) + OpenReview (CREAM-RAG)
> 获取时间: 2026-05-24
> 类型: T3 学术论文 | 幻觉缓解研究

## 核心内容摘要

两篇 2025 年关于 RAG 系统中幻觉缓解的前沿研究论文，分别从微调和自奖励强化学习角度提出解决方案。

## 关键数据与论点

### Finetune-RAG (arXiv 2505.10792)
- **核心理念**: 首个使用"不完美检索场景"训练数据的 RAG 微调方法
- **效果**: 事实准确性提升 **21.2%**（相比基线模型）
- **创新点**: 构建了模仿真实世界不完美的 RAG 训练数据集（法律/科学/书籍/网页），每条数据配对真实和虚构内容
- **Bench-RAG**: 提出的 LLM-as-Judge 评估流水线，在现实不完美检索场景下压力测试模型
- **关键洞察**: LLM 难以识别人类显而易见的上下文线索（如同名公司财务报告、日期过时信息）

### CREAM-RAG (AAAI 2026)
- **核心理念**: 一致性正则化自奖励 RAG 框架
- **效果**: 在推理和事实性基准上平均提升 **35.04%**（相比基线 LLaMA-2-7B）
- **方法**: 整合检索模块 + Actor-Critic 生成器 + DPO 自奖励强化学习 + 一致性正则化损失
- **解决的问题**: 自奖励 RAG 中奖励信号不稳定的问题（reward hacking 和 retrieval-blind collapse）
