# RAG from Scratch with LangChain and LangGraph

**来源**: GitHub - ChidambaraRaju/RAG-from-scratch-with-langchain-and-langgraph
**链接**: https://github.com/ChidambaraRaju/RAG-from-scratch-with-langchain-and-langgraph
**类型**: 完整实战教程（10+ Jupyter Notebook）
**适用章节**: Ch3, Ch4, Ch5, Ch6, Ch7, Ch12

## 课程结构

### Section 1: Data Ingestion and Parsing（数据摄入与解析）
- 01_Data_Ingestion.ipynb：TXT 文本加载
- 02_Parsing_PDF_Documents.ipynb：PDF 解析与清洗
- 03_Parsing_Word_Documents.ipynb：Word 文档解析
- 04_Parsing_Structured_Data_CSV_and_Excel.ipynb：表格数据处理
- 05_Parsing_JSON_Data.ipynb：JSON 数据处理
- 06_Parsing_SQL_Databases.ipynb：关系型数据库抽取

### Section 2: Vector Embeddings（向量嵌入）
- 01_HuggingFace_Embeddings.ipynb：开源模型 + 余弦相似度
- 02_OpenAI_Embeddings.ipynb：商业模型 + 语义搜索

### Section 3: Vector Stores（向量存储）
- 01_Traditional_RAG_with_Chromadb.ipynb：首个端到端 RAG
- 02_Building_RAG_with_FAISS.ipynb：FAISS 高性能检索
- 03_Pinecone_VectorDB.ipynb：云原生向量库

### Section 4: Advanced Chunking（高级分块）
- 1_Semantic_Chunking.ipynb：基于语义边界的分块

### Section 5: Hybrid Search（混合检索）
- 1_Hybrid_Search-Dense_Sparse.ipynb：稠密+稀疏检索
- 3_Reranling.ipynb：重排序
- 5_MMR.ipynb：最大边际相关性

### Section 6: Query Enhancement（查询增强）
- 1_Query_Translation.ipynb：查询翻译
- 2_Multi_Query.ipynb：多查询
- 3_Query_Rewriting.ipynb：查询重写
- 4_HyDE.ipynb：假设文档嵌入

### Section 7: Advanced RAG（高级 RAG）
- 1_Adaptive_RAG.ipynb：自适应检索
- 2_Corrective_RAG.ipynb：自我修正检索
- 3_Self_RAG.ipynb：自我反思生成
- 4_Fusion.ipynb：多源融合

### Section 8: Agentic RAG（智能体 RAG）
- 使用 LangGraph 构建自主 RAG Agent
- 工具调用、多步推理、自我修正

## 核心价值

1. **官方权威**：LangChain AI 推荐教程
2. **渐进式学习**：从基础数据处理到高级 Agentic RAG，8 个 Section 递进
3. **代码完整**：所有 Notebook 可直接运行
4. **技术覆盖全**：涵盖 Chunking、Embedding、Hybrid Search、Reranking、Query Translation、Agentic RAG
